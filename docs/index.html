Quick Start

Choose your language:

Python:

pip install -e ./python
python python/scripts/example_analysis.py

Julia:

julia --project=julia
julia julia/scripts/run_example.jl

MATLAB:

addpath('matlab');
run('matlab/scripts/run_example.m');
